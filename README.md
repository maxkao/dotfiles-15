# dotfiles

## Installation

### Prepare

- Install zsh

    - macOS
    ```bash
    $ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    $ brew install zsh
    ```

    - Linux
    ```bash
    $ sudo apt-get update && sudo apt-get install -y zsh
    ```

### Actual

```bash
$ curl -L raw.github.com/paveg/dotfiles/master/install.sh | bash
```
